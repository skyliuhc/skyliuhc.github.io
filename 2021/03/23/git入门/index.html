在idea中使用

如果嫌弃输入命令麻烦可以配合idea的git插件使用

先在perferences找到git ，可以直接搜索git，按照图中的顺序，先点开1，然后再选择git的安装位置

如果不知道可以使用which git来找，然后点击右边的Test

如果配置成功的话，按了测试之后会出现版本号

注意看git的三个按钮，

从左至右，第一个是从远程仓库下载代码，第二个是commit ，第三个是push，上传到远程代码

写完代码之后就可以先勾✅，然后写入commit信息，在点击最右边的push，又能上传到github啦
